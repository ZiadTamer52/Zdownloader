## Quick Download (Windows)

Download the latest **standalone .exe** — no Python installation required!

→ Go to [**Releases** → Latest version](https://github.com/ZiadTamer52/Zdownloader/releases/latest)  
→ Click on `Zdownloader.exe` (or `Zdownloader_vX.Y.Z.exe`) under **Assets**  
→ Run it directly

Antivirus note: Some AVs flag brand new unsigned .exe files → you may need to click "More info → Run anyway". This is common for self built tools.

## Features

- Download in **Best**, **4K**, **1080p**, **720p** or **Audio only (MP3 192kbps)**
- Real time progress + speed + ETA
- Video thumbnail preview (after "Get Info")
- Modern look powered by **ttkbootstrap** (dark themes)
- Custom save folder (defaults to `Downloads/YouTube Downloads`)
- Clean log console
- Ctrl+V paste + Enter to download
- Threaded → GUI stays responsive


## How to Get the .exe on GitHub Releases

1. Go to the repository → **Releases** tab (right side or [[https://github.com/ZiadTamer52/Zdownloader/releases](https://github.com/ZiadTamer52/Zdownloader/releases/latest)]
2. Click the **latest release** (e.g. `v1.0.0`)
3. Scroll to **Assets** section
4. Download `Zdownloader.exe` (or similar name)
5. Double click to launch... done
